== Description ==

= A great, must have and missing feature in WordPress  =

Is there someone else logging in to your account? Keep an eye on your recent login records.

This awesome plugin will help you see your latest login records including time of login, IP address, street address, city, country, location map, browser and OS info.

= Features  =

* It adds a 'Last Login' column in user manager page for admin, that helps admin to monitor user activities!
* Lists up to 50 of recent login records in user's profile page, that includes user's IP address, street address, location map, operating system and browser info!


== Installation ==

1. Upload the `recent-logins` folder and its contents to your `wp-contents/plugins` directory.
2. Activate in the `Plugins` menu.
3. You are done!

== Screenshots ==

1. 'Last Login' column in users manager page for admin.
2. List of recent login records in user's profile page.
3. Thickbox Pop-up to show detailed information about a specific login record.
